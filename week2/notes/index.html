JavaScript Novice to Ninja Chapters 1-4

Sets create a data structure that is full of unique values. Even if you add in duplicate values, they will be ignored.

To add data into sets, you can do it in the array given when you declare the set. You can also use the .add() method to add data into a set.

The only way to add whole strings into a set is to use multiple .add() method. If you try to add it in the array given when declaring the set, it will instead add all the letters of the word, ignoring all of the duplicate letters.

You can find the amount of items in a set by using the size() method.

You can find out if a value is in a set by using the has() method.

Sets do not use index notation.

You can delete items from a set using the delete() method.

Use the separator operator to convert a set into an array.

Default parameters go at the end of the arguments.

A callback is a function passed to another function as an argument.
