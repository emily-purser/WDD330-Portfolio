HTML5 & CSS3 For the Real World - Chapter 8 Transforms and Transitions

Transform -> Similar to using position: relative. However, with transform, you move the elements withouth impacting the documents flow. A translation can also only be translated in relation to its current position.

To transform up along the y axis, it would be translateY(-30px). To move an element right, it would be translateX(45px).

Scaled elements will default to scale from the center. If you wish to change this, use the transform-origin property.

If you want to add multiple transforms to an element, declare transform once and the separate the different transforms by a space separated list.

200ms is the optimum time for a transition.

HTML5 & CSS3 For the Real World - Chapter 12 Canvas, SVG, and Drag and Drop

When creating a canvas element, you need to specify width and height in the declaration. This is because this determines how large a canvas's coordinate system is.

Canvas is best used when you need to paint pixels and don't care about retrieving or modifying the shapes you make.

SVG is best used when you want to retrieve and modify those shapes.
